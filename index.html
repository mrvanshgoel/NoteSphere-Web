Tech Stack

Under the Hood

NoteSphere is built on a modern, robust Client-Server architecture utilizing the latest technologies.

  • Android Java Native client with Material Design Components.
  • Node.js + Express Backend REST API hosted on Render.
  • Firebase Authentication & Firestore NoSQL Database.
  • Google Gemini Custom AI Router orchestrating multiple models.

Development Status

Transparency is key. Here's where we are and where we're going.

✅ Current Status

  • Secure Auth & Profile Bootstrapping
  • File upload and AI extraction (Ephemeral Storage)
  • AI Chat with context attachment
  • Flashcard & Quiz Generators
  • Context Menus (Rename/Delete/Share)

🚧 Challenges Overcome
